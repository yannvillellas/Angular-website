# Build the app

## Install the packages

In the front and in the backend folder type :

```powershell
npm install
```

## Running the app

### Back-end

In the backend folder run this command :

```powershell
node app.js
```

### Front-end

In the front folder run this command :

```powershell
ng serve
```

The app should be running here : http://localhost:4200/
